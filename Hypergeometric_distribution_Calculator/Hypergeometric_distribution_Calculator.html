超幾何分佈計算機





超幾何分佈機率:

公式

超幾何分佈的概率可以通過以下公式計算：

P(X = k) = (C(K, k) * C(N - K, n - k)) / C(N, n)

其中：

  • C(a, b) 是組合數，計算方法是 C(a, b) = a! / (b! * (a - b)!)
  • N 是總數量
  • K 是總成功數量
  • n 是抽取數量
  • k 是抽到成功數量

計算方法

1. 計算 C(K, k)：從 K 個成功數量中選擇 k 個成功的組合數

2. 計算 C(N - K, n - k)：從剩餘的 N - K 個失敗數量中選擇 n - k 個失敗的組合數

3. 計算 C(N, n)：從總共 N 個數量中選擇 n 個的組合數

4. 使用公式計算機率 P(X = k)
